# 🌳 정글 밸런스 게임
> 사소한 고민들을 **밸런스 게임** 형태로 공유할 수 있는 사이트  
게임과 댓글로 동기들과 소통하며 자신만의 정글 리추얼을 확립해 보세요!

## 프로젝트 개요
- 개발 기간: 2024.08.05.-2024.08.08. (약 3일)
- 개발 인원: 3명
